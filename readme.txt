Компиляция и запуск:
ant compile
ant run

Общие требования:
1.       Сборка проекта и запуск main метода реализовать в виде ant-скрипта.
2.       Результаты работы выложить в github. Прислать ссылку.
3.       В письме прислать набор команд для сборки и запуска приложения
4.       jdk 1.8

Задание следующее:
№1 Реализовать класс автопарковка. А также main метод, демонстрирующий поведение.
Только алгоритм. Многопоточность реализовывать не нужно. Тесты реализовывать не нужно.

1.       Парковка имеет вместимость n машин
2.       У парковки есть стоимость 1-го часа стоянки.
3.       В ночное время с 23 до 6 часов, стоимость стоянки двойная.
4.       Для парковки есть 2 метода – вьезд и выезд
5.       Метод въезд – на входе id-машины(int), время въезда в часах(int). Результат – Boolean, может машина въехать или нет.
6.       Метод выезд – на вход id-машины(int), время выезда в часах(int). на выходе стоимость стоянки.
7.       Параметр время вьезда и выезда работает так – 0 это начало первых суток. 24 это начало(12 часов ночи) вторых суток, 73 – это час ночи четвертых суток и т.п.
